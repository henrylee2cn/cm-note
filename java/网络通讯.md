## SNI(server name indication):实现多域名主机的SSL/TLS认证
```
    在2006年之前,一台服务器只能有一个数字整证书,因为客户端发送信息不包含服务器的域名,所以当时的服务要实现SSL/TLS则必须
一台服务器只能包含一个网站,不然服务器会分不清该向客户端分配哪一个网站的数字证书.
    在2006年,TLS加入了SNI,允许客户端向服务器发送域名.
```
